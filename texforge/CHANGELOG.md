# Changelog - texforge Skill

## Version 1.0 (2026-03-31)

### Initial Release

First version of the texforge skill, created from the texforge v0.1.0 CLI project.

#### What This Skill Covers

- **All CLI commands**: `new`, `build`, `fmt`, `check`, `template` with flags and examples
- **Project structure**: `project.toml` format, directory layout generated by `texforge new`
- **Templates**: available templates, how to install and use them, compatibility with tectonic
- **Error handling**: clean error format, when to use `check` vs `build`
- **Typical workflow**: end-to-end from project creation to PDF output

#### Files Created

- `SKILL.md` — core skill with frontmatter, commands, and workflow
- `LICENSE` — MIT license
- `CHANGELOG.md` — this file
- `references/COMMANDS.md` — full command reference with all flags
- `references/TEMPLATES.md` — template structure, variables, compatibility notes
